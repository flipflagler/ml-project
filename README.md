# ML-Practicals

A collection of machine learning practical exercises and implementations covering fundamental ML concepts and algorithms.

## Project Contents

This repository contains the following Jupyter notebooks:

- **encoding.ipynb** - Data encoding techniques (one-hot encoding, label encoding, etc.)
- **pre_processing.ipynb** - Data preprocessing and preparation for ML models
- **linear_regression.ipynb** - Linear regression implementation and examples
- **logistic_regression.ipynb** - Logistic regression for classification problems

## Getting Started

### Prerequisites
- Python 3.7+
- Jupyter Notebook or JupyterLab
- Required libraries: pandas, numpy, scikit-learn, matplotlib

### Installation

1. Clone the repository:
```bash
git clone https://github.com/flipflagler/ml-project.git
cd ml-project
```

2. Install required packages:
```bash
pip install pandas numpy scikit-learn matplotlib jupyter
```

3. Start Jupyter:
```bash
jupyter notebook
```

## Usage

Open any of the `.ipynb` files in Jupyter Notebook to view and run the practical exercises.

- Data Preprocessing
